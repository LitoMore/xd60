# XD60

Configs and tools for my XD60 keyboard

## Dependency

- [dfu-programmer](https://github.com/dfu-programmer/dfu-programmer)

## Commands

```bash
$ kbd reflash         # Reflash with XD60.hex file
$ kbd reflash <path>   # Reflash with another HEX file
```
